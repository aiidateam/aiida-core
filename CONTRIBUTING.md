# Contributing to AiiDA

The AiiDA team appreciates contributions from a wide range of backgrounds.
Small improvements to the documentation or minor bug fixes are always welcome.

For the full developer guide, including how to set up your development environment, coding style, testing, and making pull requests, please see the [Developer Guide](https://aiida-core.readthedocs.io/en/latest/developer_guide/index.html) in the AiiDA documentation.

You can also browse the guide locally at [`docs/source/developer_guide/`](docs/source/developer_guide/).
