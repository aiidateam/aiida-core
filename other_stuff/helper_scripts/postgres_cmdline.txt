run
  psql -U postgres -W -h localhost

and then execute (REMEMBER TO PUT THE ; AT THE END OF THE LINE, OR THE COMMAND DOES NOT GET EXECUTED!)

# to delete the old database
  DROP DATABASE aiidadb;

to create a new database:
  CREATE DATABASE aiidadb;
or
  CREATE DATABASE aiidadb OWNER <role name>;

to get the list of databases:
  select datname from pg_database;

or directly from bash:
  psql -U postgres -W -h localhost -l
